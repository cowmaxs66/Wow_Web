## 归档规则
- 每个正式阶段完成后新增对应发布说明。
- 发布资料必须能从本 README 直接找到。
- 源码仓库不归档任何大漠二进制、授权文件、签名私钥、真实账号和商业脚本。
- P32 起允许总包和 Client 分包按明确需求携带 `dm.dll/RegDll.dll`，但不得携带授权文件、账号资料和商业脚本。
- 二进制输出只允许保存在本机 ignored 目录或明确发布的 Release 资产中，不进入 Git 源码。
