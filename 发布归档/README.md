# 发布归档说明

本目录保存发布记录、交付包说明、版本归档和交接资料。

## 当前状态
P7 阶段已建立 v1.1.0 Web 管理端信息扩展归档。本版本不产出二进制发布包，只归档可验证源码、运行说明、交接材料和安全边界。

## 当前文件
| 文件 | 职责 |
|------|------|
| `v1.0.0-发布说明.md` | 说明 v1.0.0 已包含、未包含和发布形式 |
| `v1.1.0-发布说明.md` | 说明 v1.1.0 Web 信息扩展范围和验证结果 |
| `交接清单.md` | 面向后续维护和继续开发的交接入口 |
| `部署与运行指南.md` | 记录本地构建、启动、联调和 DmBridge 编译方式 |
| `安全边界与私有资料说明.md` | 明确禁止入库资料、当前安全能力和生产前置项 |
| `最终验证报告.md` | 记录 P6 最终构建、测试和烟测结果 |

## 归档规则
- 每个正式阶段完成后新增对应发布说明。
- 发布资料必须能从本 README 直接找到。
- 不归档任何大漠二进制、授权文件、签名私钥、真实账号和商业脚本。
- 二进制输出只允许保存在本机 ignored 目录，不进入 GitHub。
