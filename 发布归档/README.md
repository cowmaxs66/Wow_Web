## 归档规则
- 每个正式阶段完成后新增对应发布说明。
- 发布资料必须能从本 README 直接找到。
- 不归档任何大漠二进制、授权文件、签名私钥、真实账号和商业脚本。
- 二进制输出只允许保存在本机 ignored 目录，不进入 GitHub。
