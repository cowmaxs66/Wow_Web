## 本版已包含
- Rust workspace：`shared-types`、`client-agent`、`management-server`。
- Client Agent：TOML 配置读取、Lua 文件加载、指令上限、结构化日志、状态输出。
- Lua 宿主：白名单 API 注册、脚本权限控制、DmBridge 高层 API。
- DmBridge：Delphi Win32 DLL 工程、C ABI 契约、STA Worker 队列、最小大漠链路。
- Management Server：健康检查、Client 状态上报、Client 状态查询。
- Web Admin：Vue/Vite 管理端 MVP，显示 Server 健康、Client 列表、状态详情和连接设置。
- 脚本安全门：manifest、Ed25519 签名、SHA-256 完整性校验、权限白名单。
- 发布资料：交接清单、部署运行指南、安全边界说明和最终验证报告。

## 本版未包含
- 生产鉴权、TLS、账号体系和权限角色。
- 远程脚本下发、脚本仓库、签名工具链和私钥管理后台。
- 数据库存储、历史日志、数据分析和客户端详细设置扩展。
- WebSocket 实时推送、命令下发和结果回传。
- 大漠全量接口、受限资料、授权文件、真实脚本和商业配置。

## 发布形式
当前版本采用源码归档，不产出可直接分发的二进制包。原因如下：
- `dm.dll`、`RegDll.dll`、CHM/CHW 文档、授权资料不允许进入仓库。
- DmBridge 需要本机 Delphi/RAD13 与大漠运行环境配合编译验证。
- 脚本签名私钥不能进入 GitHub，发布侧签名流程需要独立保护。
