# v1.1.0 发布说明

## 发布信息
| 项目 | 内容 |
|------|------|
| 版本 | v1.1.0 |
| 日期 | 2026-07-06 |
| 发布类型 | Web 管理端信息扩展 |
| 发布范围 | P7 Web 管理端快照分析、客户端详情、本地设置 |

## 本版已包含
- `ClientStatus` 协议扩展，增加 `runtime`、`script`、`server` 三类真实状态摘要。
- Client Agent 状态映射扩展，状态上报包含框架版本、系统、架构、进程 ID、脚本安全门、允许权限和 Server 上报目标。
- Web Admin 快照分析面板，展示在线比例、脚本分布、最新上报和安全门统计。
- Web Admin 客户端详情面板，展示 Agent 运行、脚本设置、Server 上报和协议 JSON。
- Web Admin 本地设置面板，管理浏览器本地 Server URL、查询 Client ID 和刷新动作。
- `shared-types` 增加 README，明确协议职责和扩展约束。

## 本版未包含
- 历史数据库、趋势图、长期在线率统计。
- 远程修改 Client Agent 配置。
- 脚本下发、命令执行、日志流、账号体系、鉴权和 TLS。
- CPU、内存、窗口、账号、角色、地图等尚未上报的客户端字段。

## 验证摘要
- `cargo fmt --all --check` 通过。
- `cargo clippy --workspace -- -D warnings` 通过。
- `cargo test --workspace` 通过，client-agent 21 项、management-server 6 项、shared-types 2 项。
- `npm run build` 通过。
- Server/Client 上报烟测通过，API 返回 `release_version = v1.1.0`、`security_enabled = true`、`report_target = 127.0.0.1:18087/api/client/status`。
- Playwright fallback 桌面 1440x920 与移动 390x844 通过，无横向溢出。

## 注意事项
修改状态协议后，直接运行旧的 `target/debug/client-agent.exe` 会得到旧协议输出。发布烟测前必须执行：

```powershell
cargo build --workspace
```
