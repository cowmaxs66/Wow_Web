# v1.10.0 发布说明

## 发布信息
| 项目 | 内容 |
|------|------|
| 版本 | v1.10.0 |
| 日期 | 2026-07-07 |
| 发布类型 | 自动更新自替换 |
| 发布范围 | `--update-apply`、独立 updater 脚本、托盘安装更新入口 |

## 本版已包含
- `client-agent-core.exe --update-apply`：检查 GitHub latest、比较版本、下载新版并安排自替换。
- 托盘菜单新增“安装更新并退出”。
- 独立 updater 脚本会停止安装目录下旧进程、备份旧文件、保留 `data/logs/updates`，再覆盖新发布包。
- 更新失败会写入 `logs/update-apply.log`。
- 版本比较避免把旧版本当成更新安装。

## 本版未包含
- MSI / MSIX 安装器。
- 发布包签名验证。
- 增量补丁。
- Service 提权更新。
- 多机器灰度发布。
