# v1.11.0 发布说明

## 版本主题
服务端远程更新与 Web Admin 导航补全。

## 主要更新
- Web Admin 左侧导航改为真实功能页：总览、客户端、脚本、远程操作、设置。
- 新增远程操作页，可从 Server 直接下发 Client 本机命令。
- Server 命令白名单新增 `update.apply`，用于触发 Client 自替换更新。
- Client monitor 新增 `update.apply` 执行分支，复用 v1.10.0 updater。
- Server 最新状态查询增加 60 秒心跳窗口，超时未上报的 Client 会显示为离线。
- 切换 Client 时历史趋势会立即刷新，不再等待下一次全局刷新。
- 新增脚本配置页，集中查看 Lua bootstrap、安全门和权限配置。

## 使用方式
1. 双击 `management-server.exe` 打开 Server 和 Web Admin。
2. 双击 `client-agent.exe` 启动客户端托盘与 monitor。
3. 在 Web Admin 左侧进入 `远程操作`。
4. 选择目标 Client 后点击 `安装更新`，Server 会写入 `update.apply` 命令队列。
5. Client monitor 下次轮询到命令后，会检查 GitHub Release、下载新版包并安排本机自替换。

## 风险说明
- `update.apply` 会触发 Client 本机更新安装流程，可能停止当前 monitor 并重启客户端。
- 当前 Server 命令队列仍为内存队列，未提供送达确认、执行回执、鉴权审计。
- 在线状态超时阈值当前固定为 60 秒，后续阶段应改为配置项。
