# v1.12.0 发布说明

## 版本主题
服务端上线日志与 Server/Client 分包。

## 主要更新
- Management Server 接收 Client 状态上报后，控制台会打印 `Client 上线`、`Client 状态刷新` 或 `Client 离线`。
- 控制台日志包含 `client_id`、`online`、`script`、`release_version`、`message_id`，便于排查客户端是否真的上线。
- 打包脚本新增 Server 分包：`WoW_Server_v1.12.0_windows.zip`。
- 打包脚本新增 Client 分包：`WoW_Client_v1.12.0_windows.zip`。
- 保留总包：`WoW_Framework_v1.12.0_windows.zip`，继续兼容当前用户安装器和 Client 自动更新。

## 使用说明
### 查看 Server 控制台上线日志
根目录 `management-server.exe` 是 GUI launcher，不显示控制台。需要看控制台日志时运行：

```powershell
.\bin\management-server-core.exe --no-open-browser
```

Client 上报后，控制台会出现类似：

```text
Client 上线 client_id=local-dev-client online=true script=bootstrap release_version=v1.12.0 ...
```

### 分包选择
| 包 | 用途 |
|----|------|
| `WoW_Framework_v1.12.0_windows.zip` | 完整测试包、当前用户安装器、自动更新兼容 |
| `WoW_Server_v1.12.0_windows.zip` | 只部署 Management Server |
| `WoW_Client_v1.12.0_windows.zip` | 只部署 Client Agent |

## 风险说明
- GUI launcher 不显示控制台，这是无控制台正式入口的预期行为；看日志需运行 `bin/management-server-core.exe`。
- Client updater 仍下载 `WoW_Framework` 总包，不下载 Server/Client 分包，避免半更新。
- 分包仍不包含 `dm.dll`、`RegDll.dll`、授权文件、真实账号资料、真实商业脚本和 JSONL 运行数据。
