# v1.13.0 发布说明

## 版本主题
Client 分包直启、离线状态收敛与远程目标选择修正。

## 主要更新
- Client 分包和总包的发布配置默认启用本机 Server 上报：`127.0.0.1:18080/api/client/status`。
- Client 发布配置的 DmBridge 路径修正为包内 `dm-bridge/Win32/DmBridge.dll`。
- 托盘和通知 PowerShell 宿主改为隐藏启动，减少双击 Client 时出现黑色控制台窗口。
- Client monitor 正常退出、Service 停止或托盘停止 monitor 时会主动上报 `online=false`。
- Server 心跳兜底窗口由 60 秒收敛为 30 秒。
- Web Admin 远程操作面板新增下发目标选择，可选择单台 Client 或全部已上报 Client。

## 使用说明
### Client 分包正式入口
双击 Client 分包根目录：

```text
client-agent.exe
```

该入口会启动托盘 UI，并由托盘拉起 monitor。默认连接本机 `127.0.0.1:18080` 的 Management Server。

### 远程操作目标选择
进入 Web Admin 左侧“远程操作”，在“下发目标”中选择：

| 目标 | 行为 |
|------|------|
| 单台 Client | 只写入该 Client 的消息或命令队列 |
| 全部已上报客户端 | 对列表中每台 Client 分别写入队列 |

## 风险说明
- 任务管理器强杀、断电或网络中断无法保证立即上报离线；Server 会在 30 秒兜底窗口后显示离线。
- 远程操作仍只确认“写入命令队列”，没有执行回执；实际执行结果以 Client 本机日志为准。
- 本版本仍不包含登录鉴权、命令审计、发布包签名验证、MSI/MSIX 安装器。
