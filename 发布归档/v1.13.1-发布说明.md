# v1.13.1 发布说明

## 发布定位
v1.13.1 是 Client 正式直启热修复版本，专门修复用户从 Client 分包根目录双击 `client-agent.exe` 后无任何效果的问题。

## 修复内容
- 新增统一 PowerShell 脚本写入工具，生成托盘、设置窗口和更新脚本时写入 UTF-8 BOM，兼容 Windows PowerShell 5.1。
- 托盘宿主和设置窗口启动 PowerShell 时显式使用 `-STA`，满足 WinForms/NotifyIcon 的线程模型要求。
- GUI launcher、通知、托盘和更新脚本启动链路只保留 `CREATE_NO_WINDOW`，避免 `DETACHED_PROCESS` 造成子进程无反馈退出。
- 打包脚本和 DmBridge 编译脚本转换为 UTF-8 BOM，避免中文源码路径在 Windows PowerShell 5.1 下乱码解析。
