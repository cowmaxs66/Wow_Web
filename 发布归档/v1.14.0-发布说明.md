# v1.14.0 发布说明

## 发布定位
v1.14.0 是 Server 托盘与双端图标版本：Server 正式入口进入右下角托盘菜单，Server/Client 发布包都使用用户提供的图标资源。

## 新增内容
- 新增 `assets/icons/` 图标资源目录，归档 Server/Client SVG 源文件和运行时 `.ico`。
- Server 新增 `management-server-core.exe --tray` 托盘入口。
- 根目录 `management-server.exe` 改为启动 Server 托盘，托盘会拉起 `management-server-core.exe --no-open-browser` 并打开 Web 管理页。
- Server 托盘右键菜单新增：
  - 启动 Server
  - 关闭 Server
  - 重启 Server
  - 打开 Web 管理页
  - 打开日志目录
  - 仅退出托盘
  - 退出托盘并关闭 Server
- Client 托盘加载 `assets/icons/client.ico`。
- 发布包复制对应图标资源：总包包含双端图标，Server/Client 分包只包含自身图标。

## 安全边界
- 未包含 `dm.dll`。
- 未包含 `RegDll.dll`。
- 未包含 CHM/CHW 文档、授权文件、真实账号、真实商业脚本、签名私钥和 JSONL 运行数据。
- 本版本只做用户态 Server 托盘，不新增 Server Windows Service、服务恢复策略、鉴权审计或命令执行回执。
