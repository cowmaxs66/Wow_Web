# v1.15.0 发布说明

## 发布定位
v1.15.0 是产品化控制中心与安装体验版本：`WoW-Manager.exe` 从单一安装入口升级为本机控制中心，让普通用户通过一个窗口完成安装、启动、日志和卸载操作。

## 新增内容
- 新增 `installer/manager-current-user.ps1` 控制中心。
- `WoW-Manager.exe` 启动控制中心，而不是直接执行安装脚本。
- 控制中心提供：
  - 安装 / 修复
  - 启动 Server
  - 启动 Client
  - 打开 Web 管理页
  - 打开日志目录
  - 打开程序目录
  - 刷新状态
  - 卸载程序
- 安装脚本支持从已安装目录执行修复，不会误删自身。
- 安装后新增 `WoW Framework` 桌面和开始菜单快捷方式。
- Server/Client 快捷方式绑定对应图标。

## 安全边界
- 未包含 `dm.dll`。
- 未包含 `RegDll.dll`。
- 未包含 CHM/CHW 文档、授权文件、真实账号、真实商业脚本、签名私钥和 JSONL 运行数据。
- 本版本不生成 MSI/MSIX，不写入系统级安装目录，不自动安装 Server Windows Service，不处理代码签名证书。
