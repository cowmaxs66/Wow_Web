# v1.16.0 发布说明 - Web 使用体验与 DM/Lua 操作流

## 版本定位
v1.16.0 面向产品化易用性：降低 Web 管理端操作复杂度，让 Client 列表、数据分析、设置说明、DM 模式和 Lua 脚本使用流程更接近正式软件。

## 新增能力
- Web Admin 客户端列表支持搜索、在线/离线/DM 权限/脚本筛选。
- Web Admin 快照分析增加健康分、DM 权限、架构分布、上报开关和风险提示。
- 设置向导改为标准操作流程，PowerShell 命令默认隐藏到进阶排错区。
- 新增 DM/Lua 使用流程面板，说明 Client 如何读取 TOML、校验 manifest、执行 `scripts/bootstrap.lua`、套用 `dm.access`。
- 新增 Server 远程命令白名单 `script.run_bootstrap`。
- Client monitor 支持执行 `script.run_bootstrap`，只重跑本机已配置并通过安全校验的 bootstrap。
- Web 常见 1280 宽度下客户端列表优先全宽显示，移动端列表保持卡片布局。

## 安全边界
- 本版本不允许 Server 下发任意 Lua 文本。
- `script.run_bootstrap` 只执行 Client 本机已有脚本，仍受 manifest、hash、签名和权限白名单约束。
- 本版本不包含大漠二进制、授权文件、私有脚本、真实账号和 JSONL 运行数据。
- MSI/MSIX、代码签名、企业级自动升级、命令回执和鉴权审计仍属于后续阶段。
