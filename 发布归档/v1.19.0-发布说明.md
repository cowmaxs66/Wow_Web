# v1.19.0 发布说明 - Client 远程配置下发

## 版本定位
v1.19.0 补齐 Client 正式配置下发闭环：Server 可下发白名单配置补丁，Client 写回本机 TOML，monitor 下一轮自动重载，Web Admin 提供可操作设置面板。

## 新增能力
- 新增远程命令 `config.apply`。
- 新增共享协议 `ClientConfigPatch`。
- Server 校验 `config.apply` payload 非空且不超过 4000 字符。
- Client 新增配置补丁写回模块，只允许修改：
  - Server 上报设置。
  - Lua bootstrap 设置。
  - 脚本安全门配置。
  - DmBridge 路径。
- Client monitor 每轮重载配置，读取失败时保留上一轮有效配置。
- Web Admin 设置页新增 Client 远程设置面板，可对单台 Client 下发配置。
- 远程操作回执可识别 `config.apply`。

## 安全边界
- 不允许远程修改 `client.id`。
- 不下发任意 Lua 文本。
- 不提交签名私钥、真实脚本、账号资料、大漠二进制和授权文件。
- 当前仍未加入登录鉴权、操作者身份、持久化审计和防重放；生产联网前必须补齐。
