# v1.20.0 发布说明 - Client 原生设置表单化

## 版本定位
v1.20.0 将 Client 本机设置窗口从 TOML 文本编辑器改为结构化 WinForms 表单。普通用户通过输入框、复选框和权限勾选完成设置；高级用户仍可打开配置文件排错。

## 新增能力
- `client-agent.exe --settings-window` 改为表单 UI。
- 表单分组包含基础、Server 上报、Lua 脚本、脚本安全门和 DM Bridge。
- 保存前校验必填字段、整数范围、Server Port、Status Path 和 Ed25519 公钥格式。
- 保存时自动创建配置目录，再写回标准 `client-agent.toml`。
- 保留“打开配置文件”按钮作为高级排错入口。
