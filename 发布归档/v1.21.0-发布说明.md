# v1.21.0 发布说明 - DM 实机烟测与多机通讯规划

## 版本定位
v1.21.0 提供可控的 DM Lua 烟测入口，用于验证 Lua -> Rust -> DmBridge -> 大漠 COM 的最小链路。同时沉淀多机器管理和通讯效率优化规划。

## 新增能力
- 新增 `scripts/dm_smoke.lua`。
- 新增 `scripts/dm_smoke.manifest.json`。
- 新增 `scripts/README.md`，说明 DM smoke 切换方式和公钥。
- 新增脚本安全测试，验证 DM smoke 的 hash、签名和权限。
- 新增 `技术设计/DM实机烟测与多机通讯规划.md`。
- 风险记录新增 DM 本机 COM 依赖和多机轮询请求量风险。

## DM smoke 行为
- 读取 DmBridge ABI 版本。
- 初始化大漠 COM。
- 读取大漠版本。
- 读取屏幕 `(0, 0)` 颜色。
- 读取最后大漠错误码。
- 关闭 DmBridge。
- 不点击、不键盘输入、不绑定窗口。
