# v1.28.0 发布说明

## 版本定位
`v1.28.0` 是远程设置与脚本日志回执版本，重点解决控制面板不能直接完成 Client 设置、Lua 脚本输出难以查看的问题。

## 已包含
- Web Admin：
  - 远程操作页直接显示 Client 远程设置面板。
  - Client 设置面板新增快捷模板：DM 内部测试、仅状态上报、停止 Lua、安全校验模式。
  - 命令回执按类别展示执行摘要、脚本输出、脚本日志和错误摘要。
- Client Agent：
  - Lua `log()` 会被纳入本次脚本执行报告。
  - 远程 `script.deploy_bundle` 推送后立即执行时，回执包含 Lua `return` 与 `log()`。
  - 远程 `script.run_bootstrap` 和 `script.start` 的回执也包含 Lua 输出。

- `cargo fmt --all --check`：通过。
- `cargo test --workspace`：通过。
- `cargo clippy --workspace -- -D warnings`：通过。

- `tools/package-release.ps1`：通过，三类 Windows zip 已生成。
- Client 分包 smoke：通过，`bin/client-agent-core.exe --run-once` 输出 `release_version = v1.28.0`、`arch = x86`、Lua 启用、脚本安全关闭和 `dm.access`。
- Server 分包 health smoke：通过，临时端口 `/health` 返回 `{"status":"ok"}`。
- Server 分包命令队列 smoke：通过，`script.deploy_bundle` 可写入并取出命令队列。
- 包内容检查：通过，总包和 Client 分包包含 `DmBridge.dll`、`dm.dll`、`RegDll.dll`；Server 分包不包含这三个 DM 文件。
- EXE 图标 smoke：通过，Client/Server 根 EXE 和 core EXE 均可读取 32x32 关联图标。

## 边界
- 命令回执仍是 Server 内存记录，Server 重启后近期回执会丢失。
- 回执只展示截断后的脚本日志，长日志仍应查看 Client 本地日志窗口。
- 快捷模板只是填表，不会自动套用到 Client。
