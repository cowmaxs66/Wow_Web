# v1.29.0 发布说明

`v1.29.0` 是桌面控制台 UI 体验优化版本，重点解决控制面板流程分散、日志入口不直观、移动端导航挤压的问题。

## 已包含
- Web Admin 侧栏改为桌面控制台风格，并新增独立“日志”导航。
- 顶部栏新增当前目标卡片，显示选中 Client、在线状态、分组标签和运行模式。
- 总览页改为 Client 列表优先，统计分析和历史趋势后置。
- 新增“日志与回执”页面，集中 Server 审计和命令回执入口。
- Client 列表空状态区分加载中、无 Client 上报和筛选无结果。
- 移动端导航改为 3 列两行，避免中文竖排和横向溢出。

## 验证结果
- `npm run build`：通过，Web Admin 版本为 `1.29.0`。
- `cargo test --workspace`：通过，Client 55 项、Server 52 项、shared-types 11 项、launcher 3 项测试全部通过。
- `tools/package-release.ps1`：通过，已生成总包、Server 分包和 Client 分包。
- Client 分包 smoke：通过，`bin/client-agent-core.exe --run-once` 输出 `release_version = v1.29.0`、`arch = x86`、Lua 启用、脚本安全关闭和 `dm.access`。
- Server 分包 smoke：通过，`/health` 返回 `ok`，内嵌首页 HTTP 200，`script.deploy_bundle` 命令可写入。
- 浏览器桌面 QA：1280px 视口无横向溢出。
- 浏览器移动 QA：390px 视口无横向溢出，导航文字不竖排。
- 浏览器控制台错误数：0。
