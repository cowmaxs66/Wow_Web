# v1.3.0 发布说明

## 发布信息
| 项目 | 内容 |
|------|------|
| 版本 | v1.3.0 |
| 日期 | 2026-07-06 |
| 发布类型 | 生产持久化 MVP |
| 发布范围 | JSONL 历史持久化、启动回放、普通编译包 |

## 本版已包含
- Management Server 可通过 `MANAGEMENT_SERVER_HISTORY_PATH` 启用 JSONL 历史持久化。
- Client 状态上报成功后会追加保存到 JSONL 文件。
- Server 启动时会回放 JSONL，恢复最新状态和历史查询 API。
- 未配置持久化路径时保持 P8 的纯内存行为。
- 普通编译包包含 Windows x64 Server/Client 可执行文件、Web Admin 静态文件和运行说明。

## 本版未包含
- SQLite/Postgres 真实数据库接入。
- 自动数据清理、压缩、轮转和长期聚合报表。
- 鉴权、TLS、生产 CORS、账号体系、审计、日志流和远程脚本下发。
- 大漠 `dm.dll`、`RegDll.dll`、CHM/CHW、授权文件和真实商业脚本。
