# v1.4.0 发布说明

## 发布信息
| 项目 | 内容 |
|------|------|
| 版本 | v1.4.0 |
| 日期 | 2026-07-06 |
| 发布类型 | 一键运行与首次设置向导 |
| 发布范围 | Server Web 托管、Web 设置向导、一键脚本、x86/x64 运行边界 |

## 本版已包含
- Management Server 可通过 `MANAGEMENT_SERVER_WEB_DIR` 托管 Web Admin 静态文件。
- Web Admin 新增首次设置向导，保存浏览器本地配置并生成启动命令。
- 新增 `tools/` 一键运行脚本。
- 发布包准备包含 x64 Server、x64 Client、x86 Client、Web Admin 和工具脚本。
- Web 向导明确区分 x64 核心模式和 x86 DM 模式。

## 本版未包含
- Windows Service、托盘程序和后台守护。
- 设置向导直接写入 Client 配置文件。
- 远程配置下发、账号体系、鉴权、TLS 和生产 CORS。
- 大漠 `dm.dll`、`RegDll.dll`、CHM/CHW、授权文件和真实商业脚本。

## 验证摘要
- `cargo fmt --all --check` 通过。
- `cargo clippy --workspace -- -D warnings` 通过。
- `cargo test --workspace` 通过。
- `npm run build` 通过。
- Management Server Web 托管烟测通过：首页 HTTP 200，`/health = ok`。
- Client 上报烟测通过，API 返回 `release_version = v1.4.0`。
- Web Admin 桌面与移动浏览器烟测通过，无页面横向溢出。
- 首次设置向导联动通过，端口修改后可自动刷新看板并显示 `local-dev-client`。
- 包内 x64 Server/Client 烟测通过。
- 包内 x86 Client 基础启动通过，输出 `arch = x86` 和 `release_version = v1.4.0`。
- 编译包敏感文件检查通过，未包含 `dm.dll`、`RegDll.dll`、CHM/CHW、`.env`、JSONL、PDB、DCU 和 MAP 文件。

## 注意事项
x86 DM 模式需要用户自行准备 32 位大漠运行环境。本发布包不包含第三方大漠二进制和授权资料。
