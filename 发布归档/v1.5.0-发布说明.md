# v1.5.0 发布说明

## 发布信息
| 项目 | 内容 |
|------|------|
| 版本 | v1.5.0 |
| 日期 | 2026-07-07 |
| 发布类型 | 单 exe 与客户端监控基础 |
| 发布范围 | Server 内嵌 Web、Client monitor、本地日志、Server 消息、通知气泡 |

## 本版已包含
- `management-server.exe` 可内嵌 Web Admin 静态资源。
- `client-agent.exe` 新增 `--monitor`、`--setup`、`--open-log`、`--notify`。
- Server 新增 Client 消息队列 API。
- Web Admin 可向选中 Client 发送消息。
- Client monitor 可轮询 Server 消息、写入本地日志，并在 Windows 下弹出通知气泡。
- Client 相对路径可从当前目录或 exe 所在目录解析，便于直接双击运行。

## 本版未包含
- 持久托盘图标和右键菜单。
- 原生设置窗口。
- Windows Service 和自动开机启动。
- WebSocket 长连接、消息送达确认和消息持久化。
- 大漠 `dm.dll`、`RegDll.dll`、CHM/CHW、授权文件和真实商业脚本。
