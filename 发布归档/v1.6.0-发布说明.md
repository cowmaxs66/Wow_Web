## 本版已包含
- `client-agent.exe --startup-status` 查询当前用户开机启动状态。
- `client-agent.exe --enable-startup` 写入当前用户开机启动项。
- `client-agent.exe --disable-startup` 删除当前用户开机启动项。
- `tools/start-client.ps1` 新增 `-StartupStatus`、`-EnableStartup`、`-DisableStartup`。
- Web Admin 首次设置向导可生成 Client 开机启动查询、启用和停用命令。
- 开机启动命令固定为当前 `client-agent.exe --monitor`。

## 本版未包含
- Windows Service。
- 完整托盘常驻图标和右键菜单。
- 原生设置窗口。
- 自动更新器。
- Server 远程下发 Client 本机设置。
- 大漠 `dm.dll`、`RegDll.dll`、CHM/CHW、授权文件和真实商业脚本。
