## 本版已包含
- `client-agent.exe --service-run` 作为真正的 Windows Service 入口。
- `client-agent.exe --service-install/start/stop/status/uninstall` 管理 `WoWClientAgent` 服务。
- `client-agent.exe --tray` 启动托盘常驻和右键菜单。
- `client-agent.exe --settings-window` 打开本机配置编辑窗口。
- `client-agent.exe --update-check/download` 检查和下载 GitHub 最新发布包。
- Server 新增 `ClientCommand` 白名单命令队列。
- Client monitor 可轮询并执行 Server 白名单命令。
- Web Admin Client 详情中可下发远程本机操作。
- `tools/run-local.cmd` 默认进入 Server + Client 托盘形态。

## 本版未包含
- 运行中自替换更新。
- 更新包签名验证、自动回滚和差分更新。
- 远程命令鉴权、审计、持久化和送达确认。
- Service 交互桌面窗口。
- 大漠 `dm.dll`、`RegDll.dll`、CHM/CHW、授权文件和真实商业脚本。
