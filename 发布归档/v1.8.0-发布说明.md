## 本版已包含
- `management-server.exe` 无参数启动 Server 并打开 Web 管理页。
- `management-server.exe --no-open-browser` / `--api-only` 保留自动化和服务化入口。
- `client-agent.exe` 无参数启动托盘常驻 UI，并由托盘拉起 monitor。
- `client-agent.exe --run-once` 保留旧的单次 JSON 输出维护入口。
- 当前用户开机启动写入 `client-agent.exe`，不再写 `client-agent.exe --monitor`。
- `tools/start-server.ps1` 在非 OpenBrowser 模式下显式传入 `--no-open-browser`。
- `tools/start-client.ps1` 增加 `-RunOnce`。

## 本版未包含
- 独立安装器。
- Windows 开始菜单快捷方式。
- 无控制台窗口 GUI launcher。
- 自动更新运行中自替换。
