# v1.9.0 发布说明

## 发布信息
| 项目 | 内容 |
|------|------|
| 版本 | v1.9.0 |
| 日期 | 2026-07-07 |
| 发布类型 | 无控制台正式入口与当前用户安装器 |
| 发布范围 | GUI launcher、当前用户安装器、正式打包脚本、开机启动 launcher 衔接 |

## 本版已包含
- 根目录 `management-server.exe` 改为无控制台 Server launcher。
- 根目录 `client-agent.exe` 改为无控制台 Client launcher。
- `bin/management-server-core.exe` 保留 Server 维护入口。
- `bin/client-agent-core.exe` 保留 x86 Client 维护入口。
- `bin/client-agent-x64-core.exe` 保留 x64 Client 维护入口。
- `WoW-Manager.exe` 支持当前用户安装和快捷方式创建。
- `WoW-Remove.exe` 支持当前用户卸载，默认保留 data/logs。
- `tools/package-release.ps1` 固化正式发布包结构。
- 开机启动优先写入根目录 `client-agent.exe` launcher。

## 本版未包含
- MSI / MSIX 安装包。
- 管理员级全局安装。
- Service 自动安装和提权。
- 自动更新运行中自替换。
- 大漠 `dm.dll`、`RegDll.dll` 注册。
