# P37 UI QA 截图说明

## 职责
本目录保存 P37 桌面控制台 UI 体验优化的浏览器验证截图。

## 文件说明
| 文件 | 说明 |
|------|------|
| `desktop-overview.png` | 1280px 桌面视口，总览页、顶部目标卡片、Client 列表优先布局 |
| `mobile-logs.png` | 390px 移动视口，日志与回执页、两行导航和远程操作入口 |

## 验证结论
- 桌面视口无页面横向溢出。
- 移动视口无页面横向溢出。
- 移动导航文字保持横排，不再被挤成竖排。
- 日志导航可进入，并能看到 Server 审计和远程操作回执入口。
