# 阶段P11-步骤表

## 阶段目标
完成 Server 单 exe、Client 单 exe 监控基础、Server 消息和 Client 本地日志/通知 MVP，形成 v1.5.0 可测试发布包。

## 完成标准
- `management-server.exe` 可在不依赖 `web-admin/dist` 外部目录时打开 Web Admin。
- `client-agent.exe --monitor` 可常驻运行、周期状态上报、轮询 Server 消息。
- Web Admin 可向选中 Client 发送 Server 消息。
- Client 可把状态与消息写入本地日志。
- Windows 下 Client 可弹出系统通知气泡。
- 发布包以 `management-server.exe` 和 `client-agent.exe` 为主要入口。
