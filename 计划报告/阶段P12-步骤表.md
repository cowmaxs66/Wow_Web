# 阶段P12-步骤表

## 阶段目标
完成 Client 本机设置与当前用户开机启动 MVP，形成 v1.6.0 可测试版本。

## 完成标准
- `client-agent.exe --startup-status` 可查询当前用户开机启动状态。
- `client-agent.exe --enable-startup` 可写入当前用户开机启动项。
- `client-agent.exe --disable-startup` 可删除当前用户开机启动项。
- `tools/start-client.ps1` 提供同等开关。
- Web Admin 首次设置向导可生成查询、启用和停用开机启动命令。
- 文档明确本阶段不做 Windows Service 和完整托盘菜单。
