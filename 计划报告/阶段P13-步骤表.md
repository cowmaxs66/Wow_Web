# 阶段P13-步骤表

## 阶段目标
完成正式运行基础：Windows Service、托盘常驻、原生设置窗口、自动更新检查/下载、Server 远程下发 Client 本机操作，形成 v1.7.0 可测试发布包。

## 完成标准
- `client-agent.exe --service-run` 可作为 Windows Service 入口。
- `client-agent.exe --service-install/start/stop/status/uninstall` 可管理 Service。
- `client-agent.exe --tray` 可启动托盘常驻和右键菜单。
- `client-agent.exe --settings-window` 可打开本机设置窗口。
- `client-agent.exe --update-check/download` 可检查和下载 GitHub 发布包。
- Server 可创建 ClientCommand，Client monitor 可轮询并执行白名单命令。
- Web Admin 可在 Client 详情中下发白名单本机操作。
