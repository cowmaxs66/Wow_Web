# 阶段P14-步骤表

## 阶段目标
完成双击正式入口：用户直接运行 `management-server.exe` 和 `client-agent.exe` 即可开始使用，不再依赖记忆命令参数。

## 完成标准
- `management-server.exe` 无参数启动 Server 并打开 Web 管理页。
- `client-agent.exe` 无参数启动托盘常驻 UI。
- `--run-once` 保留 Client 单次 JSON 输出。
- `--no-open-browser` 保留 Server 自动化运行入口。
- 开机启动默认启动 Client 托盘入口。
- 文档不再把命令行参数作为普通用户主入口。
