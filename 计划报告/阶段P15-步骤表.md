# 阶段P15-步骤表

## 阶段目标
完成无控制台正式入口与当前用户安装器：发布包根目录只暴露普通用户双击入口，维护 core exe 放入 `bin` 目录。

## 完成标准
- 根目录 `management-server.exe` 为无控制台 Server launcher。
- 根目录 `client-agent.exe` 为无控制台 Client launcher。
- `bin` 目录保留 core exe 与维护参数。
- 开机启动优先写入根目录 Client launcher。
- 发布包包含当前用户安装和卸载入口。
- 打包流程由脚本固化，避免人工遗漏文件。
