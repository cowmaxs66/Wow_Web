# 阶段P16-步骤表

## 阶段目标
完成自动更新自替换闭环：Client 可从 GitHub Release 下载新版 zip，并通过独立 updater 在旧进程退出后替换安装目录。

## 完成标准
- `--update-apply` 支持检查、下载、安排替换。
- 版本比较能避免降级和重复安装。
- updater 脚本保留 `data/logs/updates`。
- 托盘菜单提供安装更新并退出入口。
- 文档说明更新边界和风险。
- v1.10.0 编译包通过包内验证并发布。
