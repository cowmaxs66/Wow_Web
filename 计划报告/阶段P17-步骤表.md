# 阶段P17-步骤表

## 阶段目标
完成 Server 远程更新命令与 Web Admin 左侧导航补全，让日常操作从正式管理端完成，而不是依赖命令行参数。

## 完成标准
- Server 命令白名单支持 `update.apply`。
- Client monitor 收到 `update.apply` 后调用现有自替换更新器。
- Server 最新状态查询能把超时未上报 Client 自动显示为离线。
- Web Admin 左侧导航具备真实功能页：总览、客户端、脚本、远程操作、设置。
- 远程操作页可下发更新、Service、开机启动、设置窗口、日志和托盘命令。
- v1.11.0 编译包通过验证并发布。
