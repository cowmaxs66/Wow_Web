# 阶段P18-步骤表

## 阶段目标
完成 Server 控制台上线日志和 Server/Client 分包输出，让正式测试时能直接看到 Client 上线事件，并能分别下载服务端包和客户端包。

## 完成标准
- Client 上报状态后，Server 控制台打印 `Client 上线`、`Client 状态刷新` 或 `Client 离线`。
- 控制台日志包含 client_id、online、script、release_version、message_id。
- `tools/package-release.ps1` 同时生成总包、Server 分包和 Client 分包。
- Server 分包不包含 Client 配置、脚本、DmBridge。
- Client 分包不包含 Management Server 可执行文件。
- 三类 zip 都通过敏感文件检查。
- v1.12.0 编译包通过验证并发布。
