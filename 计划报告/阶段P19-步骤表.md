# 阶段P19-步骤表

## 阶段目标
修正 Client 分包正式直启体验、离线状态收敛和 Web Admin 远程命令目标选择，让 v1.13.0 可直接用于用户测试。

## 完成标准
- 双击 Client 分包根目录 `client-agent.exe` 后默认启动托盘和 monitor，并向本机 Server 上报。
- Client 托盘和通知不弹出黑色控制台窗口。
- Client 正常退出或托盘停止 monitor 后，Server 收到 `Client 离线` 状态。
- Server 心跳兜底窗口收敛为 30 秒。
- Web Admin 远程操作可明确选择单台 Client 或全部已上报 Client。
- v1.13.0 总包、Server 分包、Client 分包完成验证并发布。
