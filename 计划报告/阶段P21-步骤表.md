# 阶段P21-步骤表

## 阶段目标
接入用户提供的 Server/Client 图标，并让 Server 正式入口进入右下角托盘，提供右键快捷操作。

## 完成标准
- 图标资源归类到 `assets/icons/`，并补充目录说明。
- Server 托盘支持启动、关闭、重启、打开 Web、打开日志目录和退出操作。
- 根目录 `management-server.exe` 启动 Server 托盘，不破坏 `bin/management-server-core.exe --no-open-browser` 维护入口。
- Client 托盘加载 `client.ico`。
- 总包、Server 分包、Client 分包均包含对应图标资源。
- 三类发布包完成真实入口烟测并发布。
