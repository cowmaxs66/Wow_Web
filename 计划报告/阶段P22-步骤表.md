# 阶段P22-步骤表

## 阶段目标
把 `WoW-Manager.exe` 升级为产品控制中心，让普通用户通过一个入口完成安装/修复、启动、日志、目录和卸载操作。

## 完成标准
- `WoW-Manager.exe` 打开控制中心窗口，而不是直接静默安装。
- 控制中心提供安装/修复、启动 Server、启动 Client、打开 Web、打开日志、打开目录、刷新状态和卸载。
- 安装脚本支持从已安装目录修复，不误删自身。
- 桌面和开始菜单快捷方式包含控制中心入口，并设置图标。
- 发布包完成构建、脚本语法检查、控制中心入口烟测和 GitHub Release。
