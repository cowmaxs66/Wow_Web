# 阶段P32-步骤表

## 阶段目标

让 Client 包更接近正式 DM 使用形态：默认开启 DM 权限，发布包携带大漠运行 DLL，并让 Web 远程操作可以勾选多台 Client。

## 步骤总表

| 步骤 | 名称 | 主要产物 | 当前状态 | 备注 |
|------|------|----------|----------|------|
| P32-S01 | 默认 DM 权限 | `client-agent.toml` | 已完成 | 默认包含 `dm.access` |
| P32-S02 | DM 文件打包 | `package-release.ps1` | 已完成 | 总包和 Client 分包携带 `dm.dll/RegDll.dll` |
| P32-S03 | Client 多选操作 | `ClientRemoteActions.vue` | 已完成 | 列表框 + 复选框 |
| P32-S04 | 验证与发布 | `v1.24.0` 三类包 | 已完成 | 全量验证通过，三类包已生成，等待 Release 上传记录回写 |

## 完成标准

- Client 默认状态显示已允许 `dm.access`。
- 总包和 Client 分包包含 `dm-bridge/Win32/dm.dll` 与 `dm-bridge/Win32/RegDll.dll`。
- Server 分包不包含 DM 运行 DLL。
- Web 远程操作可勾选多台 Client 后批量下发。
- 全量验证通过，并发布 v1.24.0 GitHub Release。
