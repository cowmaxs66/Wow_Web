## 阶段目标
完成 Lua 热推送与内部测试模式：Client 默认放开脚本安全校验和已知权限，管理端可对勾选 Client 推送 Lua、启动 Lua、停止 Lua、查询 Lua 状态，并拦截短时间重复下发。

## 完成标准
- Client 默认配置支持直接运行和热推送 Lua，不再因为普通测试改脚本触发 hash 不匹配。
- Web Admin 可对勾选 Client 下发脚本包、启动、停止和状态查询命令。
- Client monitor 在 Lua 停止时仍保持在线，方便继续接收管理端命令。
- 同一批目标同一操作 15 秒内重复点击会被 Web Admin 拦截。
- Rust/Web 全量验证和三类发布包生成通过。
