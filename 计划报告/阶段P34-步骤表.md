# 阶段P34-步骤表

## 阶段目标
修复 DM 绑定诊断不足问题，新增桌面控制台入口、Client 日志查看窗口、Lua 常用接口和 Lua 接口说明文档。

## 步骤总表
| 步骤 | 名称 | 主要产物 | 当前状态 | 备注 |
|------|------|----------|----------|------|
| P34-S01 | DM BindWindow 诊断增强 | DmBridge 参数校验和错误上下文 | 已完成 | 记录 hwnd、模式和 last_dm_error |
| P34-S02 | Lua 常用接口扩展 | `safe_bind_window`、`bind_window_try`、`wait_color` 等 | 已完成 | 避免绑定失败直接中断脚本 |
| P34-S03 | Server 桌面控制台入口 | Edge App 模式桌面窗口 | 已完成 | 保留浏览器打开备用入口 |
| P34-S04 | Client 日志 UI | `--log-window` 和托盘日志窗口 | 已完成 | DPI 缩放、自动刷新 |
| P34-S05 | 验证、打包与发布 | Rust/Web/Delphi 验证、三类 zip、GitHub Release | 进行中 | 等待打包发布 |

## 完成标准
- DmBridge `BindWindow` 出错时能看到窗口句柄、绑定模式和大漠错误码。
- Lua 脚本可使用 `dm.safe_bind_window` 处理绑定失败，不再强制中断 monitor。
- Server 根入口默认打开桌面控制台窗口，而不是浏览器标签页。
- Client 托盘可打开日志查看窗口。
- Lua 接口表和使用说明已归档。
