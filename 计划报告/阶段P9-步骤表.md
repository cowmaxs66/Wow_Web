# 阶段P9-步骤表

## 阶段目标
完成 Management Server 状态历史的最小持久化能力，形成 v1.3.0 可恢复历史版本，并发布普通编译包。

## 完成标准
- `MANAGEMENT_SERVER_HISTORY_PATH` 可启用本地 JSONL 历史持久化。
- Server 重启后可从 JSONL 恢复最新状态和历史 API。
- 不配置持久化时保持 P8 纯内存行为。
- 完成 Rust、Web、Server/Client 烟测和发布包内容检查。
- GitHub 发布 `v1.3.0` Release，并上传普通编译包。
